# AI Workflow Examples

Working examples of AI workflows built with the [Business-First AI Framework](https://handsonai.info/business-first-ai-framework/).

## What's Included

### Agents

| Agent | Purpose |
|-------|---------|
| `ai-news-researcher` | Scan for the latest AI developments and news |
| `ai-productivity-researcher` | Research real-world AI productivity implementations and case studies |
| `claude-research-daily` | Daily briefing on Anthropic, Claude, and Claude Code updates |
| `hbr-editor` | Expert editorial feedback on business-focused articles |
| `hbr-publisher` | Prepare finalized articles for online publication and PDF distribution |
| `meeting-prep-researcher` | Research attendees and companies before meetings |
| `tech-executive-writer` | Translate complex technical concepts for business audiences |

### Skills

| Skill | Purpose |
|-------|---------|
| `/drafting-linkedin-posts` | Draft LinkedIn posts optimized for professional engagement |
| `/editing-hbr-articles` | Edit business articles to publication quality |
| `/extracting-article-insights` | Extract key insights from articles for content repurposing |
| `/preparing-meeting-briefs` | Prepare structured meeting briefs with attendee profiles |

### Portable Prompts

Reusable prompt templates in `prompts/` for common tasks — buyer personas, LinkedIn prospect research, and quick meeting prep.

## Installation

```
/plugin marketplace add handsonai
```

Then enable the `ai-workflow-examples` plugin.
